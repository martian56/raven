# Raven v1 docs

These pages document **Raven v1**, the original tree walking interpreter. They are kept for people who still maintain v1 code. v1 lives on the `v1.x-maintenance` branch and only receives critical fixes.

If you are writing new Raven, use **v2** instead. It is a compiled language with a different syntax (PascalCase types, no semicolons, traits and generics, sum types, goroutines, a package manager). Start at [Getting Started](../v2/guide/getting-started.md), and see the [migration guide](../v2/migration.md) if you are porting v1 code.

The v1 reference below is unchanged from the v1.6 release.

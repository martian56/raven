# Security Policy

## Supported versions

The latest release is supported for security updates.

Older versions may receive fixes at maintainer discretion, but no guarantee is made.

## Reporting a vulnerability

Please report vulnerabilities privately using GitHub Security Advisories for this repository:

- [https://github.com/martian56/raven/security/advisories/new](https://github.com/martian56/raven/security/advisories/new)

Please include:

- A clear description of the issue
- Steps to reproduce or proof of concept
- Affected version(s) and environment details
- Suggested remediation (if available)

We will acknowledge reports as quickly as possible and work on a fix and coordinated disclosure.
